1. DOM and the purpose of DOM?

The full meaning of DOM

  1. Document
  2. Object
  3. Model
The purpose of DOM

DOM is usually used to access various elements from HTML with Javascript.It is a programming interface throught which we can add, remove or modify any HTML elements by Javascript in a Website.

2. How to select HTML elements using DOM? and Name the DOM methods.

There are five ways to access HTML elements by Javascript DOM.

  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

To access all the tags of a particular tag in an HTML file together, we will used getElementsByTagName()

We will used getElementsByClassName() when we need to access multiple tags of the same type.

We use getElementById() when we need to access a particular element

When we access the first element of a selector with a css selector, we used querySelector()

When we are going to access all elements with CSS selectors, we will use querySelectorAll()

3. Event bubble

When we clicked on a Website, an event is triggered and if an event handler is set somewhere in the website, The event handler is triggered. That is event Bubbles. When adding event handlers between multiple parents and children, capture the innermost target element first. Then trigger the upper elements successively until the DOM element is reached.

4. Callback Function

A callback function is a function when another function is called as an argument. By calling that function,an operation is performed by the function from outside and is used inside the function.

Why we will used Callback Function?

Using callback functions has many advantages, including that it organizes the code and it is a powerful programming tool.
